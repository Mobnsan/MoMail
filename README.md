# MoMail 

MoMail is a modern, full-stack email marketing platform designed to help you manage contacts, create beautiful templates, and launch email campaigns with ease.

##  Features

- Dashboard: Get a bird's-eye view of your email marketing performance.
- Contact Management: Import and manage your subscribers (supports CSV/XLSX).
- Email Templates: Create and save reusable email designs.
- Campaigns: Schedule and send email campaigns to your contact lists.
- Multi-Provider Support: Switch between SMTP, SendGrid, or Mailgun for delivery.
- Dark Mode: Sleek UI with modern glassmorphism aesthetics.

##  Tech Stack

- **Frontend**: React 19, GSAP (Animations), React Router 6.
- **Backend**: Node.js, Express.
- **Security**: Bcryptjs for password hashing, Express Session for auth.
- **Email Engines**: SendGrid, Mailgun, Nodemailer.
